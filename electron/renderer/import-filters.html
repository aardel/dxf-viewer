Rule Types:
  • layer: Match DXF layer name (regex supported)
  • color: Match DXF color in RGB format (e.g., "rgb(255,0,0)" or hex "#FF0000")
  • linetype: Match DXF linetype name
  • default: Fallback rule (should be last)
