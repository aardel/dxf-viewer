Layer Mapping Wizard

Map new layers to line types and tools

1 Select Layers
2 Choose Line Types
3 Assign Tools
4 Review & Save

Step 1: Select Layers to Map

Choose which layers need mapping rules. These layers were not found in the global import filter.

Step 2: Choose Line Types

Select the appropriate line type for each layer. Line types define how the geometry will be processed.

Step 3: Assign Tools

Choose the appropriate tool for each line type. Tools define the cutting/engraving parameters.

Step 4: Review & Save

Review your mapping choices and save them to the global import filter.
