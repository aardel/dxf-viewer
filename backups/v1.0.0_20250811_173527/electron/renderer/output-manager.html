📚 Available Placeholders

Use these placeholders in your templates and custom elements:

File Information
{filename} Current DXF filename
{width} Drawing width in mm
{height} Drawing height in mm
{timestamp} Current date and time
User & Material
{user} Operator name
{material} Material type
{thickness} Material thickness
Drawing Bounds
{minX} Minimum X coordinate
{minY} Minimum Y coordinate
{maxX} Maximum X coordinate
{maxY} Maximum Y coordinate
Statistics
{count} Operation count
{entityCount} Total entities
{lineCount} Line entities
{arcCount} Arc entities
Line Numbers
{number} Current line number
{nextNumber} Next line number
